CLOVER OS FRAME

The bottom layer of Clover OS is built on blockchain. Application states are built on Polkadot network, and the storage is built on IPFS.

Clover is supported by four main agreements:

  1. Smart Contract protocol layer: supports one-stop cross-chain deployment and calls of dApp
  2. Front End protocol layer: supports front-end multi-OS adaptation and operation.
  3. Storage protocol layer: supports distributed storage of dApp data.
  4. Back End protocol layer: supports various basic Defi protocols.
